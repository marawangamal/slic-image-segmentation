## Description

SLIC - *Simple Linear Iterative Clustering* python implementation to obtain **Super-Pixels**. 
Paper: http://www.kev-smith.com/papers/SLIC_Superpixels.pdf

KMeans class also available (SLIC class inherits from KMeans)
